Проект ****Working time timer****

**Описание проекта**

Визуальный планировщик рабочих смен, который показывает прогресс выполнения в процентах.

***Основная функция:***
Рассчитывает сколько процентов рабочего времени уже отработано.
Показывает оставшееся время в виде прогресс-бара.

***Как работает:***
- Задаешь период - даты начала и окончания работы
- Указываешь часы - ежедневное рабочее время (например, 9-18)
- Запускаешь таймер - система автоматически считает прогресс

***Ключевые фичи:***
- Поддержка многодневных периодов - от одного дня до нескольких недель
- Точный расчет - учитывает только рабочие часы
- Автосохранение - прогресс не сбрасывается после перезагрузки
- Адаптивный дизайн - работает на любых устройствах
- Визуальный прогресс-бар - наглядно показывает сколько осталось

***Для кого полезно:***
Фрилансеры для отслеживания проектов
Работники со сменным графиком
Студенты для контроля учебной нагрузки
Любой, кто хочет визуализировать прогресс работы

***Скрипты npm***
- npm run start	Запуск дев-сервера Webpack для разработки с хот-релоадом
- npm run build	Сборка проекта для продакшена (Webpack создаёт папку dist)
- npm run stylelint	Проверка всех CSS/SCSS файлов на соответствие правилам Stylelint
- npm run stylelint:fix	Автоматическая корректировка CSS/SCSS по правилам Stylelint
- npm run lint	Исправление проблем с кодом JavaScript/TypeScript через ESLint
- npm run format	Форматирование кода через Prettier (JS/TS/JSON/CSS/SCSS/MD)
- npm run check	Последовательный запуск: Stylelint fix → ESLint fix → Prettier format
- npm run prepare	Настройка Husky для git-хуков (автоматически при npm install)
- npm run commit	Проверка кода через check и запуск commitizen для удобных коммитов

***Линтинг и форматирование***
- Stylelint проверяет CSS/SCSS.
- ESLint проверяет JS/TS код.
- Prettier форматирует код в едином стиле.
- Husky + lint-staged автоматически запускают эти проверки перед коммитом, чтобы предотвратить «грязный» код в репозитории.

***Конфигурации***
- Webpack настроен на работу с:
- TypeScript (.ts, .tsx)
- CSS/SCSS модулями
- MiniCssExtractPlugin для вынесения CSS в отдельный файл
- HTMLWebpackPlugin для автоматической вставки бандла в HTML
- Husky запускает pre-commit хуки.
- lint-staged проверяет только изменённые файлы.
- Prettier использует .prettierrc.js.
- Stylelint использует .stylelintrc.json.
